# Algorithms Visualizer

## This project aims to implement and visualize various search algorithms. Through an intuitive user interface, users will be able to explore and understand the functioning of classic algorithms such as:

*   **(DFS) Deep-First Search**
*   **(BFS) Breadth-First Search**
*   **(A\*) Algorith**
*   **(ACO) Ant Colony Optimization**
*   **(TSP) Traveling Salesperson Problem**

The application will allow users to interact with these algorithms, observe their solutions, and compare their performance in different scenarios.
